#### 1.  What is Shell? 
A shell is a command-line interface in Unix/Linux systems that:

1. Interprets user commands
2. Interacts with the operating system
3. Executes programs
4. Examples: bash, sh, zsh, ksh, fish, etc.

#### 2. What is Shell Scripting?
Shell scripting refers to writing a script (a file with commands) that the shell can execute in sequence.

Admins use shell scripts to: Manage users, Start/stop services and Monitor system health.
